배열 ( Array ) 
데이터를 여러개 저장 가능한 자료구조

배열 선언 방법
1. 자료형[ ] 배열명;
2. 자료형 배열명[ ];

ex)
int [ ] arr; //배열선언
arr = new int[ ]{1,2,3,4,5}; //배열 재정의

2차원 배열
다차원 배열이란 2차원 이상의 배열을 의미하며, 배열의 요소로 또 다른 배열을 가지는것을 의미합니다.
2차원 배열은 배열의 요소로 1차원 배열을 가지고,
3차원 배열은 배열의 요소로 2차원 배열을 가지게 됩니다.

2차원 배열 선언
int [ ][ ] arr = new int [크기][크기];
